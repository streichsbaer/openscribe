# Project Overview

OpenScribe is a native macOS menu bar app that turns speech into text. This section covers product direction, specifications, and how to contribute.

## Current scope

- Menu bar recording workflow with global hotkeys.
- Local and cloud speech-to-text providers.
- Optional polish step with configurable rules, providers, and models.
- Session history browser with replay and re-processing.
- Usage statistics and activity tracking.

The [Product Spec](spec.md) is the canonical reference for current behavior. The [Roadmap](roadmap.md) tracks what is planned and in progress.
